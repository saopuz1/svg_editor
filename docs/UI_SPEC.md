# 编辑器 UI 规格

本文档定义当前编辑器页面的目标结构、区域职责、交互规则与状态表现。
目的不是约束视觉细节到像素级，而是确保后续由人或 AI 实现页面时，布局与交互方向保持一致。

## 页面总览

当前页面采用五段结构：

1. 顶部菜单栏
2. 工具状态栏
3. 左侧工具栏
4. 中央画布工作区
5. 右侧属性/规则面板
6. 底部状态栏

整体风格：

- 白色为主
- 浅灰背景
- 黑灰文字
- 蓝色作为强调色

## 布局结构

### 1. 顶部菜单栏

位置：
- 页面顶部固定一行

当前菜单分组：
- 文件
- 编辑
- 视图

#### 文件

必须包含：
- 导入 SVG
- 导入 CoreDraw（可暂未实现，但入口保留）
- 导出 SVG
- 导出 JSON
- 导出生产图纸（可暂未实现，但入口保留）

#### 编辑

必须包含：
- Undo
- Redo
- 新增矩形
- 新增文本
- 删除选中

#### 视图

必须包含：
- 元素显示开关
  - 未标记
  - 车线
- 标注显示开关
  - 车线编号
  - 区域
  - 档位
  - 单双
  - DML

### 2. 工具状态栏

位置：
- 顶部菜单栏下方

必须展示：
- 当前状态，例如“就绪”
- 当前工具 ID
- 当前选中数量
- 画布尺寸

可选能力：
- 工具栏显示模式切换（图标 / 放大）

### 3. 左侧工具栏

位置：
- 主工作区左侧

必须支持：
- 收起 / 展开
- 图标模式
- 放大模式（显示名称）

工具分类：
- 选择工具
  - 单选/框选
  - 划线选择
  - 控制点编辑
- 绘图工具
  - 创建曲线
  - 创建文本

交互规则：
- 同一时刻只允许一个工具激活
- 当前激活工具必须有明显视觉态
- 点击工具后立即切换 `activeToolId`

### 4. 中央画布工作区

位置：
- 页面中间最大区域

组成：
- 棋盘/点阵背景工作区
- 中央纸张容器
- 内嵌 Fabric Canvas

必须支持：
- 左侧工具栏收起按钮
- 右侧面板收起按钮
- 画布尺寸随文档数据更新
- 当前选区与画布选区同步

### 5. 右侧面板

位置：
- 主工作区右侧

必须支持：
- 收起 / 展开
- 拖拽改变宽度
- Tab 切换

当前 Tab：
- 属性
- 全局规则栈

#### 属性 Tab

无选中对象时：
- 显示明确空状态，例如“请在画布中选择对象”

选中对象时：
- 展示基础属性
  - ID
  - X
  - Y
- 按图形类型展示属性
  - rect：填充、描边
  - textbox：文本、字号
- 展示业务类型切换
  - 未标记
  - 车线
  - 标注

按业务类型进一步展示：
- 标注：
  - 字段
- 车线：
  - 尺数
  - 是双数

#### 全局规则栈 Tab

必须包含：
- 添加规则按钮
- 清空规则入口
- 规则列表

每条规则必须支持：
- 显示规则类型
- 显示简短 ID
- 启用/停用
- 上移
- 下移
- 删除

空状态：
- 显示“暂无规则”

### 6. 底部状态栏

必须展示：
- 当前文档名称
- 当前文档版本
- 当前选中数量

## 核心交互规则

### 选区

- 选区来自 `EditState.selection`
- 画布变化会回写到编辑态
- 右侧属性面板跟随当前首个选中对象
- 删除选中按钮依据当前选区启用/禁用

### 工具切换

- 工具切换不应写入文档状态
- 工具切换不进入历史
- 工具切换应立即影响光标与画布交互模式

### 属性修改

- 所有可持久化属性修改都必须转为命令
- 不允许直接改 `editor.data`
- 当前若采用 `onChange` 提交，后续可优化为 `onBlur` 或 debounce

### 导入导出

- 导入失败必须显示错误提示
- 导出失败时可不提示，但不应产生损坏数据

## 状态规范

### 空状态

必须明确展示的空状态：
- 无选中对象
- 规则列表为空

### 禁用态

以下情况应弱化或禁用：
- 没有历史时禁用 Undo
- 没有 future 时禁用 Redo
- 没有选区时禁用删除选中

### 错误态

以下错误必须有用户可见反馈：
- SVG 导入解析失败
- 文件格式不支持

## AI 页面实现约束

- 不要改变当前总体布局分区
- 不要把业务逻辑直接塞回页面组件
- 页面只负责展示状态和触发命令
- 如需新增面板，优先保持和当前 Tabs / Section 结构一致

## 后续可扩展区

后续新增页面能力时，优先从这些位置扩展：

- 左侧：图层面板
- 右侧：更细的 Inspector 分组
- 顶部：快捷命令入口
- 底部：缩放比例、坐标信息、性能状态
